
## Prompt 02 - Best time slot for a new volunteer
Prompt 02 — New Volunteer Best Slot

文件：prompts/02-new-volunteer-best-slot.md

Purpose

Provide decision support on where a new volunteer can safely reduce operational risk.

Input
Input	Description
Risk flags & skill gaps	From Prompt 01
Abstracted volunteer profile	Skills, availability, readiness (Onboarding / Stable / Trusted)
Task

Recommend up to 2 time slots

Explain which skill gap is reduced

Suggest one safe starting task

Output
Output	Description
Recommended slots	Max 2
Skill gap covered	Skill type only
Starting task	Low-risk, observable
Prompt
You are assisting a volunteer coordinator.
Provide decision support only.

Using the risk flags and skill gaps:
- Identify up to 2 time slots where the volunteer can reduce risk.
- Match only on skills and availability.
- Use readiness as a gate, not a score.
- Suggest one safe onboarding task per slot.

Do not create rosters or rank volunteers.
